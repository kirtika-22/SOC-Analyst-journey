# Day 04 - Interview Questions

## Q1. What is a Computer Network?

**Answer:**
A computer network is a collection of interconnected devices that communicate with each other to share data, resources, and services.

---

## Q2. What are the advantages of a Computer Network?

**Answer:**
- Resource Sharing
- Fast Communication
- File Sharing
- Centralized Management
- Remote Access

---

## Q3. What is the difference between LAN, MAN, and WAN?

**Answer:**

| LAN | MAN | WAN |
|-----|-----|-----|
| Covers a small area | Covers a city | Covers large geographical areas |
| High speed | Moderate speed | Lower speed than LAN |
| Example: Office Network | Example: City Network | Example: Internet |

---

## Q4. What is the difference between LAN and WLAN?

**Answer:**
LAN uses wired connections (Ethernet cables), whereas WLAN uses wireless communication (Wi-Fi).

---

## Q5. What is Network Architecture?

**Answer:**
Network Architecture is the design and structure of a computer network. It defines how devices, services, and communication protocols work together to enable communication.

---

## Q6. What is the difference between Client-Server and Peer-to-Peer Architecture?

**Answer:**

**Client-Server**
- Centralized server
- Better security
- Suitable for enterprises

**Peer-to-Peer**
- No dedicated server
- Devices share resources directly
- Suitable for small networks

---

## Q7. What are the three Communication Modes?

**Answer:**
- Simplex
- Half-Duplex
- Full-Duplex

---

## Q8. Explain Simplex, Half-Duplex, and Full-Duplex.

**Answer:**

**Simplex:** One-way communication only.

**Half-Duplex:** Two-way communication but only one device transmits at a time.

**Full-Duplex:** Two-way communication where both devices transmit simultaneously.

---

## Q9. What are Network Transmission Types?

**Answer:**
- Unicast
- Broadcast
- Multicast

---

## Q10. What is the difference between Unicast, Broadcast, and Multicast?

**Answer:**

**Unicast:** One sender → One receiver

**Broadcast:** One sender → All devices

**Multicast:** One sender → Selected group of receivers

---

## Q11. What is Physical Topology?

**Answer:**
Physical topology describes the physical arrangement of devices, cables, and network connections.

---

## Q12. What is Logical Topology?

**Answer:**
Logical topology describes how data flows through the network regardless of the physical layout.

---

## Q13. Name the common Wired Network Topologies.

**Answer:**
- Bus
- Star
- Ring
- Mesh

---

## Q14. Name the common Wireless Network Topologies.

**Answer:**
- Infrastructure
- Ad Hoc
- Wireless Mesh

---

## Q15. Which network topology is commonly used in organizations?

**Answer:**
Star Topology.

---

## Q16. What is a Router?

**Answer:**
A router connects different networks and forwards data packets based on IP addresses.

---

## Q17. What is the difference between a Hub and a Switch?

**Answer:**

**Hub**
- Broadcasts data to all connected devices.

**Switch**
- Sends data only to the intended device using MAC addresses.

---

## Q18. What is a Firewall?

**Answer:**
A firewall is a security device that monitors and filters incoming and outgoing network traffic based on predefined security rules.

---

## Q19. What is an Access Point?

**Answer:**
An Access Point (AP) provides wireless connectivity to devices and connects them to a wired network.

---

## Q20. What is a Modem?

**Answer:**
A modem converts digital signals into analog signals and vice versa for communication over Internet service provider networks.

---

## Q21. What is DHCP?

**Answer:**
DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses and other network configuration details to devices.

---

## Q22. What is VoIP?

**Answer:**
VoIP (Voice over Internet Protocol) is a technology that allows voice communication over IP networks.

---

## Q23. What is the OSI Model?

**Answer:**
The OSI Model is a seven-layer reference model used to understand how data travels across a network.

---

## Q24. Name the seven layers of the OSI Model.

**Answer:**
1. Application
2. Presentation
3. Session
4. Transport
5. Network
6. Data Link
7. Physical

---

## Q25. What is the TCP/IP Model?

**Answer:**
The TCP/IP Model is a four-layer networking model used on the Internet.

Layers:
- Application
- Transport
- Internet
- Network Access

---

## Q26. What is an IP Address?

**Answer:**
An IP address is a logical address that uniquely identifies a device on an IP network.

---

## Q27. What is a MAC Address?

**Answer:**
A MAC address is a unique physical hardware address assigned to a Network Interface Card (NIC).

---

## Q28. What is the difference between an IP Address and a MAC Address?

**Answer:**

| IP Address | MAC Address |
|------------|-------------|
| Logical Address | Physical Address |
| Can change | Usually permanent |
| Used for routing | Used within local network |

---

## Q29. What is a Network Protocol?

**Answer:**
A network protocol is a set of rules that governs how devices communicate and exchange data over a network.

---

## Q30. Why is Networking important for a SOC Analyst?

**Answer:**
Networking helps SOC Analysts:
- Analyze network traffic
- Investigate suspicious IP addresses
- Detect malicious communication
- Understand firewall logs
- Analyze DNS requests
- Investigate security incidents
- Identify network-based attacks
