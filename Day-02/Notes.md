# Day 02 - SOC Analyst Roles & Responsibilities

## SOC Team Hierarchy

- L1 SOC Analyst
- L2 SOC Analyst
- L3 SOC Analyst
- SOC Manager

---

## Responsibilities of L1 Analyst

- Monitor SIEM alerts
- Perform alert triage
- Investigate basic alerts
- Document incidents
- Escalate real incidents to L2
- Create incident tickets

---

## Daily Workflow

1. Login and check SIEM dashboard
2. Monitor alerts
3. Perform alert triage
4. Investigate logs
5. Escalate valid incidents
6. Ticketing
7. Documentation

---

## VirusTotal Investigation

Steps:

1. Open VirusTotal
2. Search IP
3. Review detections
4. Check ASN
5. Check ISP
6. View reputation
7. Review relations
8. Check timeline
